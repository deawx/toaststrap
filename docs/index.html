Toaststrap

A simple, lightweight javascript library for creating toast popups using bootstrap 5 styles.

  • Version: 1.0.0
  • Author: Nawaf Khalifah
  • Created: 25 October 2021

If you facing and issue or want to request a feature please open a ticket via Github issues.

Installation

You can use this package directly in your browser, or use it in any JavaScript modren applicaiton.

Browser JavaScript App

Using in browser

Toaststrap uses Bootstrap 5 css styles, so before you get started be sure that you are using the latest version of Bootstrap5.

Notic It's may not work with older Bootstrap versions.
  1. First download the latest release of the library From Here
  2. Unzip the package, and copy all files to your assets folder.
  3. Update your <head> tag to be something like the below example: 
                    
<!DOCTYPE html>
<html>
<head>

<!-- bootstrap 5 -->
<link href="bootstrap.min.css" rel="stylesheet" media="all">

<!-- Toaststrap -->
<link href="assets/toaststrap.css" rel="stylesheet" media="all">
</head>
<body>
</body>
</html>
  4. Add the following lines before <body\> tag 
                    
<script src="assets/toaststrap.js"></script>
<script>
    // This is just an example
    if (window.toaststrap) {
        toaststrap({title: 'Haai', noHeader: true}).show();
    }
</script>
For more examples, you can check out the Toaststrap's Initializer

Using with modern applications

Toaststrap was developed to support modern browsers, applications built on older versions of NodeJS (CJS), and modern application (ESM).

You can find the build files inside lib folder.

  1. Install the library 
                    npm install toaststrap
  2. Import the library into your JSX component 
                    
import Toaststrap, {ToastType, ToastPosition} from 'toaststrap'
const MyComponent = (props) => {
  handleClick = () => {
    const toast = Toaststrap({
      title: 'Toast Title',
      text: 'Toast Text',
      position: ToastPosition.TOP_END,
      type: ToastType.SWEET
     });
    tost.show();
  }
}

                    npm install
  3. Start dev server 
                    npm run dev

Setup Production

  1. Clone the project 
                    git clone https://github.com/nawafscript/toaststrap.git
  2. Install packages 
                    npm install
  3. Building Toaststrap 
                    npm run prod
Information The bundle files will be inside dist folder, CJS and ESM will be inside lib folder.

Configuration

Toaststrap provides many options for customizing the toast popup.

Option Version Type Required? Default Value Description
title 1.0 string yes '' The title that will appear in the toast header.
text 1.0 string/html yes '' The content that will appear in the toast body.
position 1.0 ToastPosition yes ToastPosition.TOP_END The position where the toast will appear.
type 1.0 ToastType yes ToastType.DEFAULT The position where the toast will appear.
avatar 1.0 link no '' Custom component for showing user avatar or icon in the toast header.
parent 1.0 string no body The parent node or that container of the popups.
datetime 1.0 date no Current Date Time The human-readable time that will appear in the popup header.
duration 1.0 number yes 3000 ms The The time that the popup will take it before disappearing.
soundable 1.0 boolean no false Activate the notification sound when popup appear.
noHeader 1.0 boolean no false Hide the header.
dismissible 1.0 boolean no true Show the close button in the header.
pauseable 1.0 boolean no true When the user mouse-over on the popup, then the timer will stop and the popup will stay in the screen until the mouse-leave.
progress 1.0 boolean no true The progressbar component that will appears in the popup footer.
space 1.0 5 | 10 | 15 | 20 no 5 The margins of the popup.
soundSource 1.0 link no In broswer version the default sound will be used. But in Modren App you should set the source. Notification sound source file, and source file type should be .wav.

Methods that may be used in popup.

Method Version Type Description
