Installation

Install Toaststrap to start using it.

npm

Install Toaststrap in your Node.js powered apps with the npm package.

              
                $ npm install toaststrap

                // We recommend to use yarn
                $ yarn add toaststrap

const Toast, {ToastTypes, ToastPositions} = required('toaststrap') or import Toast, {ToastTypes, ToastPositions} from 'toaststrap' will load Toaststrap onto a Toast object.
